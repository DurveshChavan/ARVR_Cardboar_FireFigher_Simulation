Open Unity Editor with the VR_Firefighter project. In the open scene (SampleScene), perform these exact steps:

**Step 1 — PlayerRig (per D-01, D-02, D-03, RIG-01):**
1. GameObject → Create Empty → rename to "PlayerRig"
2. Set Transform position to (0, 0, 0)
3. Add Component → CharacterController
   - Height: 1.8
   - Radius: 0.3
   - Center: (0, 0.9, 0)
4. Add Component → PlayerMovement (the script from Assets/Scripts/)

**Step 2 — Main Camera (per D-04, D-05, D-06, D-07, RIG-02, RIG-07):**
1. Find the existing "Main Camera" in scene hierarchy
2. DELETE it (right-click → Delete)
3. Right-click on "PlayerRig" → Camera (creates camera as child)
4. Rename the new camera to "Main Camera"
5. Set local position to (0, 1.6, 0) — NOT world position
6. Verify tag is "MainCamera" (should auto-set)
7. DO NOT add any MouseLook, CameraController, FreeLook, or rotation scripts

**Step 3 — Test Floor (per D-17):**
1. GameObject → 3D Object → Plane
2. Set position to (0, 0, 0)
3. Set scale to (5, 1, 5)

**Step 4 — Directional Light (per D-16):**
1. Confirm the default Directional Light exists — don't change it

**Step 5 — Save:**
1. File → Save (Ctrl+S) to save the scene

**Final hierarchy should look like:**
```
SampleScene
├── PlayerRig          (CharacterController + PlayerMovement)
│   └── Main Camera    (Camera, tag: MainCamera)
├── Directional Light
└── Plane              (test floor)
```
