# 04-03: Server Room Interactive Setup

<objective>
Configure the extinguisher rack and the warning wall sign for the Server Room environment.
</objective>

<tasks>

  <task>
    <description>Add Extinguisher Rack</description>
    <read_first>
      - .planning/phases/04-server-room-environment/04-01-CONTEXT.md
    </read_first>
    <action>
      Duplicate the exact Extinguisher Rack setup from the Kitchen phase, or recreate it:
      - Navigate to the `ServerRoom` hierarchy.
      - Place the cylinders at general location `(-3.5, 0.5, 2)` near WallWest.
      - Cylinders representing extinguishers: DCP (red), CO2 (black), Water (blue). Setup identical to Kitchen.
    </action>
    <acceptance_criteria>
      - Extinguishers (or place-holder cylinders) are present near WallWest of the `ServerRoom`.
    </acceptance_criteria>
  </task>

  <task>
    <description>Build Warning Wall Sign</description>
    <read_first>
      - .planning/phases/04-server-room-environment/04-01-CONTEXT.md
    </read_first>
    <action>
      Under the `ServerRoom` root GameObject:
      1. Create a `Cube` and name it `WarningSign`.
      2. Set scale `(2.2, 0.9, 0.05)`, position `(0, 2, -3.9)` (on WallSouth).
      3. Set the material to `SignRed` (#CC0000).
      
      Add Text:
      1. Create a `TextMeshPro - Text` (3D Object -> Text - TextMeshPro) as a child of the `WarningSign`.
      2. Adjust the RectTransform so the text fits on the front face of the sign.
      3. Text content:
         "ELECTRICAL FIRE\nServer Rack Short Circuit\nDO NOT USE WATER\nCO2 ONLY"
      4. Set Font size to 4, color to white, and text alignment to center.
    </action>
    <acceptance_criteria>
      - Red `WarningSign` cube placed on WallSouth.
      - TextMeshPro child correctly displays the multi-line warning message centered and readable.
    </acceptance_criteria>
  </task>

</tasks>
