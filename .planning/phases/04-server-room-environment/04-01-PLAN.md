# 04-01: Server Room Base Geometry

<objective>
Build the fundamental Server Room environment geometry under the "ServerRoom" root GameObject using Unity primitives and existing simple solid-color materials.
</objective>

<tasks>

  <task>
    <description>Create Additional Server Room Materials</description>
    <read_first>
      - .planning/phases/04-server-room-environment/04-01-CONTEXT.md
    </read_first>
    <action>
      Ensure the following solid-color URP/Lit materials exist in `Assets/Materials/` (or similar organized folder), creating them if they do not:
      - **DarkGrey**: Hex `#2A2A2A` (matches floor)
      - **VeryDarkGrey**: Hex `#1A1A1A` (matches walls)
      - **RackGrey**: Hex `#303030` (for racks)
      - **TrayGrey**: Hex `#404040` (for cable tray)
      - **Black**: Hex `#0A0A0A` (for UPS)
      - **Green**: Hex `#00FF00` (for LEDs)
      - **Red**: Hex `#FF0000` (for LEDs)
      - **SignRed**: Hex `#CC0000` (for Wall Sign)
    </action>
    <acceptance_criteria>
      - Required color materials are created and ready for assignment.
    </acceptance_criteria>
  </task>

  <task>
    <description>Build ServerRoom Root, Floor & Walls</description>
    <read_first>
      - .planning/phases/04-server-room-environment/04-01-CONTEXT.md
    </read_first>
    <action>
      Create a new empty GameObject `ServerRoom` at `(0,0,0)` and set it to DISABLED.

      Under the `ServerRoom` root GameObject, create the following Unity primitive GameObjects with their assigned materials:
      
      1. **Floor**: Plane, scale `(1,1,1)`, position `(0,0,0)`, material `DarkGrey` (#2A2A2A)
      2. **WallNorth**: Cube, scale `(8,3,0.2)`, position `(0,1.5,4)`, material `VeryDarkGrey` (#1A1A1A)
      3. **WallSouth**: Cube, scale `(8,3,0.2)`, position `(0,1.5,-4)`, material `VeryDarkGrey` (#1A1A1A)
      4. **WallEast**: Cube, scale `(0.2,3,8)`, position `(4,1.5,0)`, material `VeryDarkGrey` (#1A1A1A)
      5. **WallWest**: Cube, scale `(0.2,3,8)`, position `(-4,1.5,0)`, material `VeryDarkGrey` (#1A1A1A)
    </action>
    <acceptance_criteria>
      - `ServerRoom` has Floor and 4 Wall children.
      - `ServerRoom` is initially inactive/disabled.
      - Transforms match exact spec values.
    </acceptance_criteria>
  </task>

  <task>
    <description>Build Server Racks and Props</description>
    <read_first>
      - .planning/phases/04-server-room-environment/04-01-CONTEXT.md
    </read_first>
    <action>
      Under the `ServerRoom` root GameObject, create the following Unity primitive GameObjects with their assigned materials:
      
      1. **Rack_Left**: Cube, scale `(0.6,2,0.8)`, position `(-1.5,1,1)`, material `RackGrey` (#303030)
      2. **Rack_Middle**: Cube, scale `(0.6,2,0.8)`, position `(0,1,1)`, material `RackGrey` (#303030)
      3. **Rack_Right**: Cube, scale `(0.6,2,0.8)`, position `(1.5,1,1)`, material `RackGrey` (#303030)
      4. **Cable Tray**: Cube, scale `(4,0.1,0.3)`, position `(0,2.1,1)`, material `TrayGrey` (#404040)
      5. **UPS Unit**: Cube, scale `(0.5,1,0.4)`, position `(3,0.5,1)`, material `Black` (#0A0A0A)
      
      **Office Chair:**
      - **Seat**: Cube, scale `(0.5,0.1,0.5)`, position `(2.5,0.5,-1)`, dark grey
      - **Back**: Cube, scale `(0.5,0.6,0.05)`, position `(2.5,0.8,-1.2)`, dark grey
      
      Add miniature Cube decorations to the front faces of the three Racks for LEDs:
      - Tiny Cubes scale `(0.05,0.05,0.05)` placed on the front face (+Z or -Z depending on rack rotation, just ensure they are visible) alternating green (#00FF00) and red (#FF0000). Place at least 2-3 per rack.
    </action>
    <acceptance_criteria>
      - 3 Racks, Cable Tray, UPS Unit, and Office Chair assembled inside `ServerRoom`.
      - Tiny LED primitives placed on the fronts of the racks.
      - Transforms match exact spec values.
    </acceptance_criteria>
  </task>

</tasks>
