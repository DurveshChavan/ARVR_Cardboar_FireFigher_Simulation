# 04-02: Server Room Fire Source

<objective>
Configure the electrical fire source on the middle server rack, complete with Particle System, collision triggers, tag, and FireController script attachment.
</objective>

<tasks>

  <task>
    <description>Add Fire Particle System</description>
    <read_first>
      - .planning/phases/04-server-room-environment/04-01-CONTEXT.md
    </read_first>
    <action>
      Create a new empty GameObject as a child of `ServerRoom/Rack_Middle` and name it `FireSource`.
      Set its local position to `(0, 2.1, 0)` relative to the rack.

      Add a `ParticleSystem` component with the following settings:
      - **Duration / Looping / Play On Awake**: 5.0, true, true
      - **Start Lifetime**: 0.5
      - **Start Speed**: 2
      - **Start Size**: 0.15
      - **Start Color**: Gradient from white (#FFFFFF) to blue (#0044FF)
      - **Emission Rate Over Time**: 50
      - **Shape**: Box (Scale: X=0.4, Y=0.1, Z=0.2)
      - **Renderer Material**: Default-Particle (or similar built-in additive material)
    </action>
    <acceptance_criteria>
      - Particle System attached to `FireSource` on `Rack_Middle`.
      - Visuals emit a blueish electrical flame effect.
    </acceptance_criteria>
  </task>

  <task>
    <description>Configure Physics and Controller API</description>
    <read_first>
      - .planning/phases/04-server-room-environment/04-01-CONTEXT.md
    </read_first>
    <action>
      On the `FireSource` GameObject:
      1. Add a `BoxCollider` component.
      2. Set `Is Trigger` to true.
      3. Set Size to `(0.6, 0.5, 0.8)`.
      4. Ensure the GameObject has the Tag `FireZone`.

      Add the `FireController.cs` script to the `FireSource` object.
    </action>
    <acceptance_criteria>
      - BoxCollider trigger spans the required area (Size 0.6 x 0.5 x 0.8).
      - Tag is accurately set to `FireZone`.
      - Script `FireController` is attached.
    </acceptance_criteria>
  </task>
