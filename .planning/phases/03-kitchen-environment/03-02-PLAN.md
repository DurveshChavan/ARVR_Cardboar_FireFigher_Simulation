# 03-02: Fire Particle System and Registration

<objective>
Configure the LPG cylinder with a looping fire particle system, a FireZone trigger collider, and a FireController script stub.
</objective>

<tasks>

  <task>
    <description>Register FireZone Tag</description>
    <read_first>
      - ProjectSettings/TagManager.asset
    </read_first>
    <action>
      Register an exact newly string "FireZone" as a custom tag in Unity's TagManager. This is necessary before assigning the tag to the fire object.
    </action>
    <acceptance_criteria>
      - `ProjectSettings/TagManager.asset` contains the `FireZone` tag entry
    </acceptance_criteria>
  </task>

  <task>
    <description>Create FireController Stub</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Create a new C# script `Assets/Scripts/FireController.cs`. This is a stub using the same pattern as `GameManager.cs` from Phase 2.
      
      Content:
      ```csharp
      using UnityEngine;

      public class FireController : MonoBehaviour
      {
          // Stub for phase 5 fire suppression logic
          public void StartSuppression()
          {
              Debug.Log("Fire suppression started on: " + gameObject.name);
          }
      }
      ```
    </action>
    <acceptance_criteria>
      - `Assets/Scripts/FireController.cs` exists and contains the placeholder method
      - Script compiles without errors
    </acceptance_criteria>
  </task>

  <task>
    <description>Configure Fire Particle System</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Create a child GameObject named "FireParticles" under the `LPGCylinder` object created in Plan 03-01. Add a Particle System component with the following exact settings:
      
      - **Start Lifetime**: 0.8
      - **Start Speed**: 1.5
      - **Start Size**: 0.3
      - **Start Color**: gradient orange (`#FF6600`) to yellow (`#FFCC00`)
      - **Emission Rate**: 30 per second
      - **Shape**: Sphere, radius 0.1
      - **Renderer Material**: Default-Particle
      - **Looping**: true
      - **Play On Awake**: true
    </action>
    <acceptance_criteria>
      - `FireParticles` object is a child of `LPGCylinder`
      - Particle System component is attached and configured exactly per settings
    </acceptance_criteria>
  </task>
  
  <task>
    <description>Configure Fire Zone Collider</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      On the "FireParticles" GameObject (or the LPGCylinder, whichever holds the fire):
      
      1. Assign the tag: `FireZone`
      2. Add a `SphereCollider` component
      3. Set `radius` to `0.4`
      4. Set `Is Trigger` to `true`
      5. Attach the newly created `FireController.cs` script.
    </action>
    <acceptance_criteria>
      - Tag is set to `FireZone`
      - SphereCollider is attached, `isTrigger == true`, `radius == 0.4`
      - `FireController` script is attached in the Inspector
    </acceptance_criteria>
  </task>

</tasks>
