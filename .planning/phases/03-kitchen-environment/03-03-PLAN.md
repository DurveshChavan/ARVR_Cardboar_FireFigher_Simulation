# 03-03: Extinguisher Rack and Wall Sign

<objective>
Assemble the extinguisher rack with three color-coded extinguisher props, rack labels, and the main informational wall sign.
</objective>

<tasks>

  <task>
    <description>Create Extinguisher Materials</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Create the following solid-color URP/Lit materials in `Assets/Materials/` (or use existing from Plan 01 if already red):
      - **Black**: Hex `#111111` (for CO2 extinguisher)
      - **Blue**: Hex `#2244CC` (for Water extinguisher)
      - **Yellow**: Hex `#FFEE00` (for Wall sign)
    </action>
    <acceptance_criteria>
      - Three Material assets exist with the specified hex colors
    </acceptance_criteria>
  </task>

  <task>
    <description>Build Extinguisher Rack</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Under the `Kitchen` root GameObject, create three Cylinders to represent the extinguishers:
      
      1. **ExtRack_DCP**: Cylinder, scale `(0.12,0.5,0.12)`, position `(-3.5,0.5,2)`, material `Red`
      2. **ExtRack_CO2**: Cylinder, scale `(0.12,0.5,0.12)`, position `(-3.5,0.5,2.4)`, material `Black`
      3. **ExtRack_Water**: Cylinder, scale `(0.12,0.5,0.12)`, position `(-3.5,0.5,2.8)`, material `Blue`
    </action>
    <acceptance_criteria>
      - Three Cylinders exist in `Kitchen` hierarchy with specified transforms and materials
    </acceptance_criteria>
  </task>

  <task>
    <description>Create Rack Labels</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Create small label Cubes above each extinguisher, and a text label reading "DCP | CO2 | Water":
      
      1. Parent the label under `Kitchen` (or a sub-object for the rack).
      2. Create small label Cubes above each cylinder: scale `(0.3,0.05,0.15)`
      3. Create text on wall nearby: use a primitive Cube root, scale `(0.8,0.3,0.05)`, with a child 3D `TMP_Text` component.
      4. The `TMP_Text` content: `DCP | CO2 | Water`
    </action>
    <acceptance_criteria>
      - Label Cubes created
      - Rack text label configured with TextMeshPro
    </acceptance_criteria>
  </task>

  <task>
    <description>Create Wall Sign</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      On the interior face of `WallSouth`:
      
      1. Create a `Cube` scale `(2,0.8,0.05)`, position `(0,2,-3.9)`, assign the `Yellow` material. This places it 0.1 units from the wall.
      2. Add a child GameObject with a 3D `TMP_Text` component.
      3. Content: 
         ```
         GAS LEAK — LPG Cylinder
         Class C Fire
         Select correct extinguisher
         ```
      4. Settings: Font size 4, Color black, Center aligned, Overflow: overflow. (Adjust RectTransform size to wrap appropriately if needed).
    </action>
    <acceptance_criteria>
      - Wall sign Cube exists with Yellow material at correct position
      - TMP_Text child is correctly configured with exactly the multiline string specified
    </acceptance_criteria>
  </task>

</tasks>
