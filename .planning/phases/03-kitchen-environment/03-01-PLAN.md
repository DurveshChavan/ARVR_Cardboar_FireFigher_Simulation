# 03-01: Kitchen Base Geometry

<objective>
Build the fundamental Kitchen environment geometry under the "Kitchen" root GameObject using Unity primitives and simple solid-color materials.
</objective>

<tasks>

  <task>
    <description>Create Kitchen Materials</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Create the following solid-color URP/Lit materials in `Assets/Materials/` (or similar organized folder):
      - **Grey**: Hex `#808080` (for floor)
      - **White**: Hex `#FFFFFF` (for walls)
      - **LightGrey**: Hex `#A0A0A0` (for counter)
      - **DarkGrey**: Hex `#303030` (for stove top)
      - **Red**: Hex `#CC2200` (for LPG cylinder)
    </action>
    <acceptance_criteria>
      - Five Material assets exist with the specified hex colors
      - Materials use standard URP/Lit shader
    </acceptance_criteria>
  </task>

  <task>
    <description>Build Kitchen Room (Floor & Walls)</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Under the existing `Kitchen` root GameObject (ensure its position is `(0,0,0)`), create the following Unity primitive GameObjects with their assigned materials:
      
      1. **Floor**: Plane, scale `(1,1,1)`, position `(0,0,0)`, material `Grey`
      2. **WallNorth**: Cube, scale `(8,3,0.2)`, position `(0,1.5,4)`, material `White`
      3. **WallSouth**: Cube, scale `(8,3,0.2)`, position `(0,1.5,-4)`, material `White`
      4. **WallEast**: Cube, scale `(0.2,3,8)`, position `(4,1.5,0)`, material `White`
      5. **WallWest**: Cube, scale `(0.2,3,8)`, position `(-4,1.5,0)`, material `White`
    </action>
    <acceptance_criteria>
      - `Kitchen` has Floor and 4 Wall children
      - Transforms match exact spec values
    </acceptance_criteria>
  </task>

  <task>
    <description>Build Counter, Stove, and Gas Cylinder</description>
    <read_first>
      - .planning/phases/03-kitchen-environment/03-CONTEXT.md
    </read_first>
    <action>
      Under the existing `Kitchen` root GameObject, create the following Unity primitive GameObjects with their assigned materials:
      
      1. **Counter**: Cube, scale `(3,0.9,1)`, position `(-2,0.45,-3)`, material `LightGrey`
      2. **Stove Top**: Cube, scale `(1.2,0.05,0.9)`, position `(-2,0.92,-3)`, material `DarkGrey`
      3. **LPGCylinder**: Cylinder, scale `(0.3,0.4,0.3)`, position `(-1,0.4,-3)`, material `Red`
    </action>
    <acceptance_criteria>
      - Counter, Stove Top, and LPGCylinder objects exist as children of `Kitchen`
      - Transforms match exact spec values
    </acceptance_criteria>
  </task>

</tasks>
