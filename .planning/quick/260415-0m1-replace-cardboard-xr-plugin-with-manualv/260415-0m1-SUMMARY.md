## Scene Hierarchy Setup (3 Steps)

### Step 1 — Build the Hierarchy

In your Unity scene, create this exact object tree under your `PlayerRig`:

```
PlayerRig                    ← CharacterController, your movement scripts
└── Neck                     ← Empty GameObject (gamepad yaw/pitch applied here)
    └── Head                 ← Empty GameObject (gyroscope rotation applied here)
        ├── LeftEye          ← Camera component
        └── RightEye         ← Camera component
```

**How to create it:**
1. Right-click `PlayerRig` in Hierarchy → _Create Empty_ → name it **Neck**
2. Right-click `Neck` → _Create Empty_ → name it **Head**
3. Right-click `Head` → _Camera_ → name it **LeftEye**
4. Right-click `Head` → _Camera_ → name it **RightEye**

**Local Positions (all start at 0,0,0):**
- Neck: `(0, 1.7, 0)` — approximate eye height
- Head: `(0, 0, 0)`
- LeftEye: `(0, 0, 0)` ← script sets IPD automatically
- RightEye: `(0, 0, 0)` ← script sets IPD automatically

---

### Step 2 — Wire ManualVRRig

1. Select **PlayerRig** in the Hierarchy.
2. Click **Add Component** → search **ManualVRRig** and add it.
3. In the Inspector, drag-assign the 4 fields:

| Field | Drag... |
|-------|---------|
| **Left Eye** | The `LeftEye` Camera |
| **Right Eye** | The `RightEye` Camera |
| **Neck** | The `Neck` GameObject |
| **Head** | The `Head` GameObject |

4. Leave **IPD** at `0.064` (64mm — standard).
5. Leave **Yaw Speed** at `90` and **Pitch Speed** at `60`.

---

### Step 3 — Camera Cleanup

To avoid rendering conflicts:

1. **Delete (or disable) the old Main Camera** that was used with the Cardboard rig.
2. On **LeftEye Camera**: ensure it has `Depth = 0`, tag = `MainCamera`.
3. On **RightEye Camera**: ensure it has `Depth = 0`, tag = `Untagged`. Remove any **AudioListener** component from it — only one AudioListener is allowed per scene.
4. Both cameras will automatically be constrained to their respective half-screen rects by the script on Start.

## Gyro Coordinate Conversion (Reference)

```csharp
// Android gyro is RIGHT-HANDED. Unity is LEFT-HANDED.
Quaternion raw = Input.gyro.attitude;
Quaternion converted = new Quaternion(raw.x, raw.y, -raw.z, -raw.w);
// Rotate -90° on X: "phone upright facing you" → "looking forward in Unity"
Quaternion unityRot = Quaternion.Euler(-90f, 0f, 0f) * converted;
```

Applied to **Head** (child of Neck) so gyro stacks on top of gamepad look direction.
